# FlagSecurePatcher(Tested on Baklava)

**Author:** Adnan  
**Module ID:** flagsecurepatcher  
**Version:** m1 (1.0.2)  

---

## Overview

**FlagSecurePatcher** is a Magisk module that allows you to **take screenshots and record your screen in apps that normally block these actions** by enforcing `FLAG_SECURE`.  

---

## Features

- Disable `FLAG_SECURE` globally
- Work with apps that prevent screenshots or screen recording
- Lightweight and easy to install via Magisk
- Compatible with Android versions 14-16(Tested) that support Magisk and Zygisk(KSU Version Willbe Coming Soon)

---

## Installation

1. Download the ZIP file.
2. Open Magisk Manager → Modules → Install from Storage.
3. Reboot your device.
4. The module will now prevent apps from blocking screenshots.

---

## Notes

- This module **modifies the behavior of apps**, so use responsibly.
- Some banking or DRM apps may detect this and block functionality.
